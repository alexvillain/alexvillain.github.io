Couvertures & mode d'emploi (50 mots)






Demande:
Faire 5 couvertures d'après 10 mots pour chacune issue de 3 procédés dont un ayant un mode d'emploi.

Request:
Make 5 covers based on 10 words each from 3 processes, one of which has a user manual.





Couvertures:
Ces 5 couvertures ont été dessinées à la main. Entre déformation/répétition/etc plusieurs styles visuels définissent les couvertures. Différents formats ont également été utilisés.

Mode d'emploi:
Le mode d'emploi est basé sur la méthode mathématique. C'est un dépliant expliquant comment trouver des mots dans un livre.
